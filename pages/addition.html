Practise addition

How many digits would you like the integers to have?

3

Try this:




+

6 8 3

2 3 4

1 11 7


+

We can solve this problem by adding together the numbers in the same column, starting from the right. If we take the example on the left, this means that we would start by adding together 3 and 4.

That adds up to 7, so let's write that underneath the problem in the same column. Now we move one column to the left, which means that we need to add together 8 and 3.

8 and 3 add up to 11, which means that, when we write the number underneath, it continues onto the column to the right, and doesn't just fit in one column. So how do we deal with this?

Instead of ignoring the number, or worse, making a new column for it, we instead leave it in that column and act as though it is above the line instead. This means that now we will be adding together 6, 2, and 1.

And now we have our final answer: 917!
